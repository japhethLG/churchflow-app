## Typography

**Font Family:** Inter (headline, body, label — all Inter)  
**Letter Spacing:** Headlines use `-0.02em` (tight/premium feel). Labels use `+0.05em` in All Caps.  
**Text Color:** Never pure black — use `on-surface` (`#191C1E`) for primary, `on-surface-variant` (`#464555`) for secondary.

| Scale | Use Case |
|---|---|
| `display-md` / `headline-lg` | Dashboard welcome, page heroes |
| `headline-md` | Section headers |
| `headline-sm` | Card headers |
| `body-md` | General body copy |
| `body-sm` | Secondary details |
| `label-md` (All Caps, +0.05em) | Category tags, metadata |
| `label-sm` | Timestamps, captions |

## Design Philosophy: "The Digital Sanctuary"

### Core Approach: Soft Minimalism + Tonal Depth
Rather than a utilitarian SaaS tool, ChurchFlow feels like a **premium printed journal** — authoritative yet warm, with elements floating in a curated light-filled environment.

### The "No-Line" Rule
Explicit borders (1px solid) are **prohibited** for sectioning. Boundaries must be created via:
1. **Background Color Shifts** — e.g., `surface-container-low` on `surface`
2. **Vertical Space** — generous whitespace separates elements
3. **Tonal Transitions** — moving between surface levels creates hierarchy

### The "Glass & Gradient" Rule
- **Hero & Primary CTAs:** Subtle linear gradient from `primary-container` → `primary` at 135°
- **Floating Overlays:** `surface-container-lowest` at 80% opacity + 20px backdrop-blur (frosted glass)

### Elevation via Tonal Layering (not shadows)
Traditional drop shadows are avoided. Instead:
- Place `surface-container-lowest` (#FFF) cards on `surface-container` (#ECEEF0) backgrounds
- **Ambient Shadows** (modals only): 32px blur, y:8px, 4% opacity **indigo tint** (not black)
- **Ghost Border**: `outline-variant` at **15% opacity** only for accessibility/high-contrast

## Components

### Buttons
| Type | Style |
|---|---|
| **Primary** | Gradient fill (`primary-container` → `primary` 135°), `xl` radius, no shadow |
| **Secondary** | `surface-container-high` bg + `primary` text |
| **Tertiary** | Plain text, `label-md`, underline on hover |

### Cards & Lists
- **No horizontal dividers in lists** — use `8px` vertical margin + hover bg transition to `surface-container-low`
- **Cards**: `lg` (1rem) radius, min `spacing-6` (1.5rem) internal padding

### Input Fields
- Background: `surface-container-highest` (no border at rest)
- Focus: transition to `surface-container-lowest` bg + 2px soft `primary` glow
- Radius: `md` (0.75rem)

## Do's and Don'ts

### ✅ Do
- Use **asymmetrical margins** for a magazine-layout feel (e.g., wider left margin on headers)
- Use `tertiary` (`#7E3000`) sparingly for "Human Touch" moments — anniversaries, prayer highlights
- Favor **white space** over filling the screen — increase type size rather than adding boxes

### ❌ Don't
- Use 1px borders — use background color changes instead
- Use standard drop shadows — use Tonal Layering
- Use high-contrast error backgrounds — use `error-container` + `on-error-container` for a "forgiving" UI
- Crowd the edges — always use `spacing-16`/`spacing-24` for page gutters
